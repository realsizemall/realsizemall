# 리사몰 (realsizemall)

리사몰 공식 웹사이트

## GitHub Pages 설정 방법

1. GitHub 저장소 페이지로 이동: https://github.com/realsizemall/realsizemall
2. **Settings** 탭 클릭
3. 왼쪽 메뉴에서 **Pages** 클릭
4. **Source** 섹션에서:
   - Branch를 `master` 또는 `gh-pages` 선택
   - Folder를 `/ (root)` 선택
5. **Save** 클릭

몇 분 후 사이트가 다음 주소에서 접근 가능합니다:
- `https://realsizemall.github.io/realsizemall/`

## 로컬 개발

```bash
# 저장소 클론
git clone https://github.com/realsizemall/realsizemall.git

# 브라우저에서 index.html 열기
```
